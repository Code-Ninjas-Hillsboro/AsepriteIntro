Intro to Aseprite!

Aseprite is an incredibly useful pixel art creation tool. It's been in development since 2001 and has been getting better and better as development continues. It's been used to make pixel art in many notable games, including Celeste, Unpacking, and Pizza Tower!

The program has a unique setup for getting it. Buying it usually costs $20, but they also allow you to download and compile the source code yourself. So what you're paying for is really just the convenience of it. Fortunately, being a company of coders, it's not too hard for us to just compile it. So it has been compiled and should be on every computer. To open it, just find the "Aseprite" program on your desktop. That should open to the following screen:

Any images you've worked on should appear in the "Recent files" section. But presumably you're looking to make a new one, and for that, you want to use the "New File" button:

There are a few settings available here. The Size section contains the width and height of the image. This can always be changed later, but it's nice to start with a good size. For the example image we will be making, we will want an image size of 32x64.
